# Computer Graphics - OpenGL

Projetos de Computação Gráfica desenvolvidos em OpenGL/GLUT (C).

## Como correr
- Abrir o ficheiro `.sln` no **Visual Studio**
- Carregar em **F5** ou no botão ▶

## Edição de código
- Editar com **VS Code**
- Para enviar alterações:
  git add .
  git commit -m "descrição"
  git push
